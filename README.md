# Corporate Chaos — Developer Excuse Generator

An AI-powered workplace incident recovery tool. Fill out a parody HR incident report, and get 5 professionally unhinged excuses generated by Groq's LLaMA 3.3 70B model to help you survive your next career-threatening moment.

---

## Features

- Bureaucratic HR form UI with incident severity, witness selection, and time-of-incident fields
- 5 AI-generated excuses per submission — each with a unique delivery tone, believability score, risk level, pro tip, and success rate
- Survival probability meter and overall situation assessment
- Framer Motion animations throughout — staggered card reveals, 3D entrance effects, animated progress bars
- Lucide React icons, Tailwind CSS v4, fully responsive

## Getting Started

### 1. Clone and install

```bash
git clone <repo-url>
cd corporate-chaos
npm install
```

### 2. Set up environment variables

Create a `.env` file in the project root:

```env
VITE_GROQ_API_KEY=your_groq_api_key_here
```

Get a free API key at [console.groq.com](https://console.groq.com).

### 3. Run the dev server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
npm run preview
```
